To execute the program, run the following in the proj0 directory in bash:
$ make
$ ./reverse <file address> <A, B, or C>

Program will return an error if file address is invalid.
A, B, and C correspond to the choice of part A, B, or C of the project description.

---------------------------------------

sample input:
The quick brown fox
Jumped over
The lazy brown
Dog

output part A:
Dog
The lazy brown
Jumped over
The quick brown fox

revo depmuJ

goD
